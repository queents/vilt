# VILT Framework

VILT stack admin panel

## Install

```bash
composer require queents/vilt
```
Now Install VILT

```bash
php artisan vilt:install
```

run NPM or YARN to install and build assets

```bash
npm i & npm run build
```

OR

```bash
yarn & yarn build
```

Generate Username and Password for Dashboard

```bash
php artisan roles:install
```
